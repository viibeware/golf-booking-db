Release Notes
0.1.7 2026-04-24 Current
  • Auto-sync update banner — every tab polls /api/version once a minute and pops a non-blocking "Update available" banner when the server's version or source build-id moves past the one the page loaded with. Triggers on same-version redeploys too (the build-id is a content hash of every .py/.html/.css/.js file), so users always know when to refresh.
  • Print Record output bumped to 14pt body text — labels, values, and tee-time slots now read cleanly on paper; headings scaled up to match.
0.1.6 2026-04-16
  • V1 Yes/No toggle as the first section of the new intake modal and edit form
  • Lodging Notes moved directly under Preferred Dates for faster capture next to the arrival date
  • New intake modal closes via X or Cancel only — backdrop clicks and Escape no longer dismiss it, preventing accidental data loss
  • Pressing X or Cancel resets the form in place (fields, tee-day rows, toggles) without reloading the page
0.1.5 2026-04-15
  • Settings moved to the sidebar footer as a gear icon next to the logout button
  • Smart date pairing — selecting an arrival date automatically sets the departure minimum to the next day and opens the departure calendar on that date
0.1.4 2026-04-13
  • Lodging Notes field under Lodging Accommodations
  • Per-course notes for Jones, Palmer, and Crispin (replaces Tee Times Notes)
  • Removed Billing column from table views
